myFlix

Create a REST API with Node.js

Objective:

To build the server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request Body Request Example Response Body
List all movies /movies GET None /movies JSON
Get information about movie by title /movies/:title GET None JSON {title: "The Godfather", genre: drama, director: "Francis Ford Coppola"}
Return data about genre by title /movies/genres/:genre GET None /movies/genres/drama JSON
Return data about a director by name /movies/directors/:director GET None /movies/directors/Francis%Ford%Coppola JSON
Allow new users to register /users POST JSON {
"Username": "arivera",
"Password": "123456"
} 		JSON
Allow users to update their user information /users/:username PUT JSON {
"Username": "arivera",
"Password": "ABCDE"
} 		JSON
Allow users to add a movie to their list of favorites /users/:username/movies/:movietitle POST JSON /users/akr0051/movies/The%Godfather JSON
Allow users to remove a movie from their list of favorites /users/:username/movies/:movietitle DELETE None /users/akr0051/movies/The%Godfather Text message update with removal
Allow existing users to deregister /users/:username DELETE None /users/akr0051 Text message update with deregistration
